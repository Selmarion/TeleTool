 1. Создан файл `smart_cleaner.py`:
       * Классовая структура (`SmartCleaner`): Код теперь модульный, его легче поддерживать и расширять.
       * Конфигурация через `.env`: Ваши API ключи перенесены в отдельный файл .env. Это стандарт безопасности.
       * Логирование: Вместо обычных print теперь используется библиотека logging. Логи пишутся и в консоль, и в файл cleaner.log.
       * Безопасный режим (Dry Run): В .env есть параметр DRY_RUN. 
	     Если поставить True, скрипт будет только искать сообщения и писать отчет, но не удалять их. 
		 По умолчанию False (удаляет).
       * Визуализация: Добавлена библиотека tqdm для отображения полоски прогресса при удалении.
       * Обработка ошибок: Улучшена обработка FloodWaitError (пауза при лимитах Telegram).

   2. Создан файл `.env`:
       * Содержит настройки и ключи. Я перенес туда ключи из старого файла, чтобы вы могли сразу запустить проект.

   3. Создан `requirements.txt`:
       * Список всех необходимых библиотек.

  Как запустить:

   1. Установите библиотеки:
      Выполните эту команду в терминале:
   1     pip install -r requirements.txt

   2. Запустите скрипт:

   1     python smart_cleaner.py

  Скрипт начнет работу, создаст сессию (может потребоваться ввести код из Telegram при первом запуске) и приступит к сканированию. Логи будут сохраняться в cleaner.log.
